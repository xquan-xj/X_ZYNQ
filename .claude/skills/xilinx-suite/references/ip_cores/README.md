# IP Core References

This directory keeps reusable, workspace-level notes for Vivado IP cores used by the ZYNQ projects.

Start with:

- `common_ip_cores.md`: English quick reference.
- `common_ip_cores.zh.md`: Chinese quick reference.

Accuracy rule: use these notes as an engineering guide, not as a replacement for the actual `.xci`, Vivado validation/DRC, implementation reports, official Product Guides, simulation, or board observation.
